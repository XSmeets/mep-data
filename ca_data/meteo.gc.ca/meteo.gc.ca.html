Comment utiliser
  • Ajouter aux raccourcis
    • Accédez à la page que vous souhaitez ajouter à vos favoris, ouvrez le menu des raccourcis météo et cliquez sur le bouton « Ajouter aux raccourcis »
    • Le nom de la page sera affiché dans la fenêtre « Ajouter cette page ». Le nom peut être modifié en mettant en surbrillance le texte et en entrant le nom souhaité
    • Cliquez sur le bouton « Sauvegarder » afin d’ajouter le lien à la liste
    • Cliquez sur le bouton « X » afin de fermer le panneau
  • Supprimer un lien
    • Cliquez sur le bouton « Organiser raccourcis »
    • Sélectionnez le lien que vous souhaitez supprimer
    • Cliquez sur le bouton « Supprimer » afin de supprimer le lien de la liste
    • Cliquez sur le bouton « Fermer » afin d’enregistrer le changement
  • Déplacer un lien dans la liste
    • Cliquez sur le bouton « Organiser raccourcis »
    • Cliquez sans relâcher le lien et déplacez le à un nouvel endroit dans la liste
    • Cliquez sur le bouton « Fermer » afin d’enregistrer le changement
  • Renommer un lien
    • Cliquez sur le bouton « Organiser raccourcis »
    • Sélectionnez le lien que vous souhaitez renommer
    • Cliquez sur le bouton « Renommer » afin de modifier le nom du lien
    • Cliquez sur le bouton « Entrée » ou le bouton de « Tab »
    • Cliquez sur le bouton « Fermer » afin d’enregistrer le changement
  • Navigation par clavier
    • Utilisez la touche de Tab afin de naviguer à un bouton ou un lien
    • Utilisez la touche Entrée pour cliquer sur les boutons ou les liens
    • Organisez la navigation des raccourcis
      • Utilisez la touche flèche vers le haut ↑ pour sélectionner le lien ci-dessus. Si aucun n'est sélectionné, le premier lien sera sélectionné.
      • Utilisez la touche flèche vers le bas ↓ pour sélectionner le lien ci-dessous. Si aucun n'est sélectionné, le dernier lien sera sélectionné.
      • Utilisez la touche Page Haut pour déplacer vers le haut une sélection dans la liste.
      • Utilisez la touche Page Bas pour déplacer vers le bas une selection.
  • Rétablir la liste par défaut

    Avertissement : en cliquant sur le bouton ci-dessous, tous vos liens personnalisés vont être supprimés.
